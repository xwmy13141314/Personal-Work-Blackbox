# Personal Work Blackbox 使用说明

## 快速启动

1. 双击 `启动.bat`（首次运行会自动安装依赖）
2. 点击「▶ 启动」按钮开始采集
3. 正常使用电脑，工具在后台静默记录
4. 需要日报时，点击「生成今日报告」

## 界面操作

| 按钮 | 功能 |
|------|------|
| ▶ 启动 | 开始采集键盘输入、窗口切换、剪贴板 |
| ⏸ 暂停 / ▶ 恢复 | 临时暂停或恢复采集 |
| ⏹ 停止 | 完全停止采集，自动导出今日日志 |
| 类型选择 | 选择报告类型：日报 / 周报 / 月报 |
| 日期选择 | 选择目标日期（周报/月报选该周期内任意一天即可） |
| 生成报告 | 调用 AI 生成对应类型的报告 |
| 查看报告 | 打开已生成的 Markdown 报告 |
| 隐私模式 | 暂停所有记录 30 分钟（处理敏感操作时使用） |

### 报告类型说明

| 类型 | 数据来源 | 输出文件 |
|------|----------|----------|
| **日报** | 单日活动数据（键盘+窗口+剪贴板） | `2026-05-22_091156_report.md` |
| **周报** | 该自然周（周一~周日）内已有的日报汇总 | `2026-05-19_weekly.md` |
| **月报** | 该自然月内已有的日报汇总 | `2026-05_monthly.md` |

> **注意**: 周报和月报基于已有的**日报**汇总生成。如果某天缺日报，该天会被标注为缺失，不影响整体生成。请确保先为需要的日期生成过日报。

## 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Alt+P | 暂停/恢复采集 |
| Ctrl+Alt+R | 生成今日报告 |
| Ctrl+Alt+N | 隐私模式 (30分钟) |

## 数据存储

- 数据库: `data/blackbox.db`（6 张表，含日报和周报/月报）
- 日志导出: `data/logs/YYYY-MM-DD.md`
- AI 日报: `data/logs/YYYY-MM-DD_HHMMSS_report.md`
- AI 周报: `data/logs/YYYY-MM-DD_weekly.md`（周一日期）
- AI 月报: `data/logs/YYYY-MM_monthly.md`

## 配置文件

编辑 `config/config.yaml` 可修改：

- **键盘记录开关**: `collection.keyboard_enabled`
- **剪贴板监控**: `collection.clipboard_enabled`
- **空闲阈值**: `collection.idle_threshold`（默认 300 秒）
- **AI 模型**: `ai.glm.model`（glm-4-flash / glm-4.5-flash / glm-4.7）
- **应用黑名单**: `privacy.app_blacklist`

## 隐私说明

- 所有数据 100% 本地存储，不上传任何服务器
- 密码管理器等黑名单应用自动跳过
- 连续数字、手机号、身份证号、邮箱自动脱敏
- 随时可点击「隐私模式」暂停记录
- 数据可随时删除（删除 data 目录即可）

## 杀毒软件白名单

如果启动时被杀毒软件拦截，请运行：
```
scripts\setup_whitelist.bat
```

## 开机自启（可选）

运行以下脚本注册开机自动启动：
```
scripts\install_startup.bat
```
